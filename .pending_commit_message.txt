equipment: Phase 13 persistence baseline (versioned schema + hash + roundtrip test)

Adds EQUIP_V1 versioned equipment serialization (line-oriented key/value) and 64-bit FNV-1a integrity hash API (rogue_equipment_state_hash). Implements serialize/deserialize covering core item attributes: base def, item_level, rarity, affix indices/values, durability, enchant level, quality, sockets (count + up to 6 gem ids), affix lock flags, fracture flag. Deserializer tolerates legacy (no header) input & skips unknown tokens for forward compatibility.

New module: equipment_persist.[ch]. New unit test: test_equipment_phase13_persistence (round-trip determinism + hash stability). Roadmap Phase 13 updated (13.1 & 13.4 Done; others Partial). C89 compliance fixes for MSVC (no mixed declarations). Lays groundwork for future migration handlers & explicit unique/set/runeword identifiers.
